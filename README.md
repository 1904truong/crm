# 🚀 CRM System - Fullstack Management Solution

Hệ thống quản lý khách hàng (CRM) tích hợp tự động hóa và thông báo qua Zalo ZNS.

## 🛠 Công nghệ sử dụng
- **Frontend:** Next.js, Tailwind CSS, Lucide Icons.
- **Backend:** NestJS, Prisma ORM, PostgreSQL.
- **Integration:** Zalo Cloud Service (ZNS).
- **DevOps:** Docker, Docker Compose.

## ✨ Tính năng chính
- **Quản lý khách hàng:** Lưu trữ thông tin, lịch sử mua hàng và tương tác.
- **Bảo hành & Sửa chữa:** Theo dõi tình trạng bảo hành (Warranties).
- **Zalo ZNS:** Gửi thông báo tự động tới khách hàng qua Zalo.
- **Automation:** Tự động hóa các quy trình chăm sóc khách hàng dựa trên quy tắc.
- **Báo cáo:** Thống kê dữ liệu kinh doanh.

## 📦 Hướng dẫn cài đặt nhanh
1. **Clone repo:** `git clone https://github.com/1904truong/crm.git`
2. **Triển khai bằng Docker:** `docker-compose up -d`
3. **Cấu hình môi trường:** Tạo file `.env` trong thư mục backend dựa trên file mẫu.
